# Calculadora TRIX

La calculadora TRIX es una herramienta desarrollada en JavaScript que permite estimar el estado trófico de ecosistemas marinos a partir de cuatro parámetros ambientales: Clorofila-a, Saturación de Oxígeno, Nitrógeno Total y Fósforo Total. Su objetivo es facilitar una evaluación rápida del grado de eutrofización de un cuerpo de agua, apoyando estudios ecológicos, programas de monitoreo y análisis de calidad ambiental.

## ¿Qué es el índice TRIX?

El TRIX (Trophic Index) es un indicador ecológico ampliamente utilizado para describir el nivel de productividad y eutrofización en ambientes marinos y estuarinos. Fue propuesto por Vollenweider et al. (1998) y combina cuatro variables clave relacionadas con nutrientes y actividad biológica. El índice se calcula mediante una función logarítmica que integra:

 - Clorofila-a

 - Saturación de oxígeno

 - Nitrógeno total

 - Fósforo total

Los valores resultantes se interpretan generalmente en cuatro categorías tróficas:

Oligotrófica: aguas con baja productividad

Mesotrófica: productividad intermedia

Mesoeutrófica: niveles elevados de nutrientes

Eutrófica: alta productividad y riesgo de degradación ambiental
